# react-common

**react-common** is the `@rtorcato` React monorepo. Its first published package,
[`@rtorcato/shadcn-ui`](https://www.npmjs.com/package/@rtorcato/shadcn-ui), is a
React component library built on shadcn/ui, Radix UI, TypeScript, and Tailwind
CSS v4, published to the public npm registry.

:::info Docs are under construction
This site is being built out — track progress in the
[react-common issues](https://github.com/rtorcato/react-common/issues).
:::

## Install

```bash
pnpm add @rtorcato/shadcn-ui
```
