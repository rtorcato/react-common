# react-common

[![CI](https://github.com/rtorcato/react-common/actions/workflows/ci.yml/badge.svg)](https://github.com/rtorcato/react-common/actions/workflows/ci.yml)
[![@rtorcato/shadcn-ui](https://img.shields.io/npm/v/@rtorcato/shadcn-ui?label=%40rtorcato%2Fshadcn-ui)](https://www.npmjs.com/package/@rtorcato/shadcn-ui)
[![@rtorcato/react-common](https://img.shields.io/npm/v/@rtorcato/react-common?label=%40rtorcato%2Freact-common)](https://www.npmjs.com/package/@rtorcato/react-common)
[![License: ISC](https://img.shields.io/badge/License-ISC-blue.svg)](https://opensource.org/licenses/ISC)

**react-common** is the `@rtorcato` React monorepo. It publishes two packages to
the public npm registry:

- [`@rtorcato/shadcn-ui`](https://www.npmjs.com/package/@rtorcato/shadcn-ui) —
  a React 19 component library built on shadcn/ui, Radix UI, TypeScript, and
  Tailwind CSS v4.
- [`@rtorcato/react-common`](https://www.npmjs.com/package/@rtorcato/react-common) —
  the foundation package: headless React hooks plus the `cn` utility and
  `ThemeProvider`, with zero UI dependencies.

## Install

```bash
pnpm add @rtorcato/shadcn-ui
```

New here? Start with [Getting started](./getting-started), then
[Tailwind setup](./tailwind-setup) and [Theming](./theming).

## Use with AI

The repo ships an in-repo skill so AI coding tools use the packages correctly
(barrel vs subpath imports, the `styles.css` + Tailwind setup, theming).

### Claude Code

Install the self-hosted skill in a Claude Code session:

```
/plugin marketplace add rtorcato/react-common
/plugin install react-common@react-common
```

### Other AI tools (Cursor / Copilot / Codex)

Point the tool at [`AGENTS.md`](https://github.com/rtorcato/react-common/blob/main/AGENTS.md)
at the repo root — it carries the same rules in the cross-tool convention many
agents read automatically.
