# react-common

**react-common** is the `@rtorcato` React monorepo. It publishes two packages to
the public npm registry:

- [`@rtorcato/shadcn-ui`](https://www.npmjs.com/package/@rtorcato/shadcn-ui) —
  a React 19 component library built on shadcn/ui, Radix UI, TypeScript, and
  Tailwind CSS v4.
- [`@rtorcato/react-common`](https://www.npmjs.com/package/@rtorcato/react-common) —
  the foundation package: headless React hooks plus the `cn` utility and
  `ThemeProvider`, with zero UI dependencies.

:::info Docs are under construction
This site is being built out — track progress in the
[react-common issues](https://github.com/rtorcato/react-common/issues).
:::

## Install

```bash
pnpm add @rtorcato/shadcn-ui
```
