# Date picker (range)

A popover two-month date range picker.

```ts
import DatePickerWithRange from '@rtorcato/shadcn-ui/components/ui-extended/date-picker-with-range'
```

<StorybookEmbed storyId="ui-extended-datepickerwithrange--default" height={420} />

## Usage

Uncontrolled (manages its own state):

```tsx
<DatePickerWithRange onChange={(range) => console.log(range)} />
```

Controlled:

```tsx
const [range, setRange] = useState<DateRange>()
<DatePickerWithRange value={range} onChange={setRange} />
```

## Props

| Prop | Type | Default | Description |
| --- | --- | --- | --- |
| `value` | `DateRange` | — | Controlled selected range. Omit for uncontrolled usage. |
| `onChange` | `(range?: DateRange) => void` | — | Called when the selected range changes. |
| `defaultValue` | `DateRange` | — | Initial range when uncontrolled. |
| `className` | `string` | — | Wrapper class. |
