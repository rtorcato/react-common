# Components

`@rtorcato/shadcn-ui` ships two kinds of components, each imported from its own
subpath (there is no package root export):

- **`components/ui/*`** — the stock [shadcn/ui](https://ui.shadcn.com) primitives,
  pre-built so you don't run the CLI per project. Documented upstream; the table
  below links each to its ui.shadcn.com page and a live Storybook story.
- **`components/ui-extended/*`** — hand-written compositions **not** on
  ui.shadcn.com. Each has its own page in this section with props + a live demo.

Every component is previewable in [Storybook](pathname:///react-common/storybook/)
(60 stories, light/dark + theme presets). Setup — importing `styles.css` and the
Tailwind `content` globs — is covered in [Tailwind setup](../tailwind-setup).

## Extended components

Hand-written, not on ui.shadcn.com — see each page for props and a live demo:

- [Confirm dialog](./ui-extended/confirm-dialog) — AlertDialog wrapper with confirm/cancel + loading
- [Data table](./ui-extended/data-table) — sortable, paginated TanStack Table
- [File upload](./ui-extended/file-upload) — drag-and-drop upload with validation
- [Multi-select](./ui-extended/multi-select) — searchable multi-value combobox
- [Page header](./ui-extended/page-header) — title + breadcrumbs + actions
- [Date picker (presets)](./ui-extended/date-picker-with-presets) — popover picker with quick presets
- [Date picker (range)](./ui-extended/date-picker-with-range) — two-month range picker
